## How to use
### Run test
- For mac or linux
```
$ ./gradlew clean test
```
- For windows
```
$ .\gradlew.bat clean test
```

### Run web
1. Start web server
    - For mac or linux
    ```
    $ ./gradlew clean bootRun
    ```
    - For windows
    ```
    $ .\gradlew.bat clean bootRun
    ```

2. Connect to `http://localhost:8080`
